# WhatsTheHash
WhatsTheHash is a script that takes a hashed string as input and tries to tell you what hash type it is. 

## Installation
```
git clone https://github.com/x10mrrobot01x/WhatsTheHash.git
cd WhatsTheHash
pip install colorama 
```
## Usage for Linux
``python wth.py``

## Usage for Windows
```
python wth.py
```
